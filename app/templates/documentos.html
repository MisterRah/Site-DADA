Introdução

O intuito desta página é justamente dar uma breve explicação sobre como fizemos o site e quais ferramentas utilizamos.

Desenvolvimento

O principal assunto que abordaremos é como foi estruturado o site:

Front-end

O front-end é basicamente toda a parte visual do site, que o usuário consegue ver. Nessa parte visual, utilizamos HTML (HyperText Markup Language), que é uma linguagem de marcação usada na construção de páginas web e que é interpretada pelos navegadores que utilizamos. Além disso, usamos CSS (Cascading Style Sheets), que é basicamente uma ferramenta que possibilita a aplicação de estilos aos elementos HTML. E, por último, mas muito importante, a linguagem de programação chamada JavaScript, que permite implementar itens mais complexos em páginas web. Por exemplo, a parte de verificação das respostas do quiz foi feita em JavaScript. Basicamente, o código em JS (abreviação de JavaScript) no quiz verifica qual alternativa o usuário selecionou e se ela está certa ou errada. Independentemente de qual seja a resposta do usuário, o código fará algo a respeito disso.

Back-end

O back-end é a parte de um sistema ou aplicação que lida com a lógica, operações, cálculos e gerenciamento de dados que ocorrem no servidor. É o "lado invisível" de uma aplicação, responsável por processar as requisições feitas pelo front-end (a parte visível e interativa com o usuário) e retornar as respostas apropriadas. No nosso caso, utilizamos um framework chamado Flask, que é utilizado para Python Web (Python é outra linguagem de programação). Pense no framework como uma "base" ou "esqueleto" para a sua aplicação. Em vez de começar do zero, você usa essa base que já tem muitas funcionalidades prontas, permitindo que você se concentre mais nas partes específicas e únicas do seu projeto. O Flask para esse site foi utilizado basicamente para o gerenciamento de rotas dentro do nosso site, que seria, basicamente, quando você clica para ir para outra página do site. Também usamos outro recurso do Flask conhecido como flash, que é basicamente uma mensagem que aparece na tela; você poderá ver isso quando entrar na página inicial ou no quiz.

Conclusão

Basicamente, esta página foi criada com a finalidade de mostrar um pouco mais sobre o site, principalmente para aqueles que podem ter ficado curiosos sobre o assunto. De qualquer maneira, se você chegou até aqui, muito obrigado pela sua atenção
